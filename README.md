# MINI PEOJECT PEMROGRAMAN VISUAL
Aplikasi ini bernama DompetKu, merupakan aplikasi yang membantu mencatat keuangan pribadi. Aplikasi ini menggunkan bahasa python dengan PySide6 dan juga SQLite3 sebagai mesin database-nya sehingga data yang ada akan disimpan di file lokal keuangan.db
Untuk menjalankan aplikasi ini, pastikan sudah menginstall PySide6 dengan cara menjalankan command berikut di terminal `pip install PySide6`
Setelah itu aplikasi dapat dijalankan dengan perintah `python main.py`
